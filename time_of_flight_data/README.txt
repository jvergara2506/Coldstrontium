These absorption images are taken at various times after releasing the atoms from the trap and allowing them to expand. There are 3 image for every time of flight that can be averaged.

File name specifications
 - Date: 06/03/2021
 - Image size: 512 x 512
 - Description: 473_odt
 - Parameters: (p=tof) time of flight parameter
 - Parameter Values: #0.??? how long in time of flight before image
 - Time: HHMMSS
 - Which image:
    _0: atom image
    _1: reference image
    _2: dark image

For example, one set of absorption image data is

2021_06_03_512x512_473_odt_(p=tof)#0.011_191240_0.bin
2021_06_03_512x512_473_odt_(p=tof)#0.011_191240_1.bin
2021_06_03_512x512_473_odt_(p=tof)#0.011_191240_2.bin

This data was taken on June 3rd, 2021 at 7:12:40 PM. The images are 512x512 pixels for the 473_odt experiment. The atoms were imaged in time of flight after falling for 11 milliseconds. _0 is the atom image, _1 is the reference image, _2 is the dark image.
